PODL Editor - Visual Editor for PODL Class Definitions

How to Use This Tool
  1. Paste your PODL content below and click "Parse PODL"
  2. Drag & Drop fields using the handle to reorder them
  3. Click "Edit" button to modify field properties, or to delete
  4. Use Fullscreen mode for easier navigation with large files
  5. View the generated output in real-time on the right panel
  6. Export by copying to clipboard or downloading the file

Edit Properties
Click "Edit" to modify:
  • Field Name - Rename the field (e.g., INTU_FLD_DURATION)
  • Field Type - Change type (INT, STRING, ARRAY, DECIMAL, etc.)
  • ORDER - Field order number
  • DESCR - Description text
  • LENGTH - Field length (for STRING/BINSTR types only)
  • CREATE/MODIFY - Access permissions
  • AUDITABLE/ENCRYPTABLE - Flags
  • SQL_COLUMN/SQL_TABLE - DB mapping
